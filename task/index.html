GeoNames Examples

The GeoNames geographical database covers all countries and contains over
eleven million placenames that are available for download free of charge.

API Name

API Description
findNearbyPlaceName Find nearby populated place using latitude and longitude
gtopo30 gtopo30 is a global digital elevation model (DEM) with a horizontal
grid spacing of 30 arc seconds (approximately 1 kilometer). GTOPO30 was derived from several
raster and vector sources of topographic information. Documentation : USGS Gtopo30 latitude and longitude
Address Geocoding Returns the latitude and longitude for a given address (may not work for all countries ! )
